## Description

Utitlity package for error. ([pkg/errors](https://github.com/pkg/errors) + alpha)


## from pkg/errors

* Annotating with stack trace
* Wrapping error

## plus alpha

* Annotating with custom error code
* multi errors to single error

### Go version

Go 1.8 or later
